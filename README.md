# 🚀 Harness CI/CD Zero to Hero - Complete Enterprise Course

## 📺 10 In-Depth Episodes (1.5-3 Hours Each)

This is a **project-based, enterprise-grade** CI/CD course using Harness.
By the end, you'll have built a complete production CI/CD platform.

## 🛠️ Technologies Covered

- **Harness**: CI, CD, GitOps, STO, OPA
- **Containers**: Docker, Docker Buildx, Multi-stage Builds
- **Kubernetes**: Deployments, Services, ConfigMaps, Secrets
- **Helm**: Charts, Values, Templates, Rollback
- **AWS**: EKS, ECS, ECR, Secrets Manager, IAM
- **Security**: Trivy, SonarQube, Gitleaks, OWASP
- **GitOps**: Argo CD, Harness GitOps
- **Monitoring**: Prometheus, Grafana, Datadog, New Relic
- **Governance**: OPA Policies, Approval Gates, RBAC
- **Notifications**: Slack, Email, Microsoft Teams

## ⭐ How to Use This Repository

1. Start from Episode 1 and go in order
2. Each episode has its own folder with:
   - `README.md` - Detailed notes (explained like you're 5)
   - `diagrams/` - Visual explanations
   - `code/` - All configuration files
   - `steps/` - Step-by-step instructions with screenshots descriptions
3. Follow along, pause, practice, repeat
